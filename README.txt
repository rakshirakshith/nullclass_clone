----------------------------------------
About Project
----------------------------------------
This project is a clone of the Nullclass website.  
I have built this using Django (Python) with HTML, CSS, and Bootstrap.  
The main goal of this project is to create a working website that looks and works like the original Nullclass site.

----------------------------------------
Main Features
----------------------------------------
- Home page with multiple sections  
- Courses page showing all available courses  
- Each course has a detail page with lessons  
- User can sign up and log in  
- Razorpay test payment added for buying courses  
- “My Courses” page shows purchased courses  
- “My Orders” page shows all payments  
- Contact page added for queries  
- Admin panel to add and manage courses and lessons  

----------------------------------------
Tools and Technologies
----------------------------------------
- Python 3  
- Django framework  
- SQLite database  
- HTML, CSS, Bootstrap  
- Razorpay (Test Mode)

----------------------------------------
How to Run
----------------------------------------
1. Open terminal and activate virtual environment  
   → venv\Scripts\activate  

2. Run the project  
   → python manage.py runserver  

3. Open this link in browser  
   → http://127.0.0.1:8000/  

4. Admin panel  
   → http://127.0.0.1:8000/admin/  
   Username: rakshith  
   Password: (your admin password)
